Trivia App

Short Info

A simple trivia website where you can answer questions and get your score.

About this project

This website contains two modes - preset question & ai generated question. There is leaderboard for preset question and those questions are based on computer science topic.

Features

  • User Authentication - signin & signup option
  • Secure Password Reset - reset password via Q&A
  • Dynamic Leaderboard - leaderboard for preset question trivia
  • AI Powered Questions - AI generated questions using Gemini API
