Tic Tac Toe

Short Info

This is XOX or Tic Tac Toe game. It is two player game, played using X and O. Player who get three consecutively.

About this project

Tic Tac Toe is a two-player game played on 3x3 grid. One player will be X and and the other player will be O. Player who gets straight line of three symbols verically, horizontally or diagonally. When a player wins game ends and when the grid fills games ends in a tie.

Features

  • Result board : shows the result
  • Reset : resets grid
