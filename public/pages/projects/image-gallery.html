Short Info

About this project

This is an image gallery in which you can view images in grid. It has a simple and light theme.

Features

  • Add : add new images to the gallery
  • Remove : remove images from the gallery
  • Replace : replace an image in the gallery with an image of your choice
  • Slideshow : preview images in the gallery and changes by time, can be changed manually also
