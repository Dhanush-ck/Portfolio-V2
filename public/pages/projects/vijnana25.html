About this project

This an official website for the vijnana 2025. Vijnana 2025 is the technical fest conducted by Kodiyeri Balakrishnan Smaraka Government College Thalassery(KBSGCT). This website includes all the details about the tech fest and the list of events conducted as part of vijnana 2025.

Features

  • Poster preview - preview of the event posters
  • Timer - show left time till the event day
  • Event filter - filter events according to participant criteria
