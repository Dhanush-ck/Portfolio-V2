Treasure Hunt

Short Info

Website for conducting treasure hunt as part of Vijñäna 2026

About this project

It is a website created to host treasure hunt. It consists of different levels, and the answers were hidden in different places of the college. This mainly focuses on smartphone users, as smartphones will be used at the time of hunt.

Features

  • Login - access the correct level through login
  • Block Users - block other users when the limit count is met
  • Restrict Levels - restrict the user from illegally entering higher level
  • Leaderboard - view level details of all users
